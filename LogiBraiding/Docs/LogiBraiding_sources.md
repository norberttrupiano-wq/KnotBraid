# Sources LogiBraiding

## Références principales
- Solid Sinnet variations.pdf (2008), auteur: Norbert.
- Traduction française: Charles Hamel (alias Nautile).
- Pentalpha #3082-2 - 78 brins.csv
- Rose des vents.csv

## Utilisation dans l'application
- Les tables CSV alimentent les mouvements des tresses #3082-2 et #3082-3.
- Le document PDF est accessible depuis l'interface via le bouton `Doc`.

## Notes de maintenance
- Conserver les rotations au format `P` / `Q`.
- Conserver les colonnes CSV dans l'ordre: etape, case, brins, depart, arrivee, rotation, note.
